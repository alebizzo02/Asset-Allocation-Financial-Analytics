# Asset Allocation – Financial Analytics

Independent multi-factor financial analysis of U.S. equities focused on valuation, risk, and portfolio construction.

## Project Overview
This project analyzes U.S. equity assets using fundamental and risk-based factors to support asset allocation decisions. The analysis combines Excel-based financial modeling with interactive Power BI dashboards to evaluate performance, volatility, and diversification effects.

## Methodology
- Factor-based analysis (valuation and risk metrics)
- Portfolio construction and comparison across scenarios
- Performance and risk evaluation using historical data
- Data cleaning, modeling, and scenario analysis in Excel
- Interactive visualization and insights in Power BI

## Tools & Skills
- Excel (financial modeling, scenario analysis)
- Power BI (dashboards, data visualization)
- Financial analysis & asset allocation
- Risk and performance metrics

## Files
- **Asset_Allocation_Model.xlsx** – Financial model and calculations
- **PowerBI_Dashboard.pdf** – Interactive dashboard export
- **Asset Allocation Analysis.pdf** – Analytical report and insights

## Notes
This is an independent project developed to demonstrate applied financial analytics and decision-making skills.
